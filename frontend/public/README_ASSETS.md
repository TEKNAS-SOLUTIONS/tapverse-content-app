# Assets Directory

Place the following files in this directory:

1. **logo.png** - Tapverse logo (for header/sidebar)
   - Recommended size: 200x60px or similar aspect ratio
   - Format: PNG with transparent background

2. **favicon.png** - Favicon for browser tab
   - Recommended size: 32x32px or 64x64px
   - Format: PNG

These assets will be automatically served from the `/` path in the application.
